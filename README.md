# mat_model_2

Десктопное приложение для математических расчётов коэффициентов ВВСТ (ВТИ, МТС) на PySide6.

## Описание

Приложение содержит 6 калькуляторов для расчёта различных показателей:

1. Коэффициент соответствия наиболее значимых показателей
2. Обеспеченность ВВСТ на год
3. Обеспеченность исправными образцами
4. Обеспеченность современными образцами ВВСТ
5. Доля современных образцов ВВСТ
6. Коэффициент технической готовности ВВСТ

## Запуск

Убедитесь, что установлен Python 3.13 и PySide6.

Запустите:

```bash
python main.py
```

## Структура файлов

- `main.py`: Основное приложение
- `component.py`: Компонент калькулятора
- `formulas.py`: Функции расчёта формул
